#Kommandoer som brukes#
|  kommando     |  beskrivelse      |
| ------------- | ----------------- |

|  git commit -m "melding"  |  Hvis du vil skrive kommentaren rett i terminalen/cmd  |
|  git push  |  Sender alle commits til github  |
|  git pull  |  Henter endringer fra github  |
